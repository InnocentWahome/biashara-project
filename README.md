# Project - (client & server)

> CompanyName System

## Installation 

```bash
### Clone the git repository
git clone
```

## Installation server

```
### Setup environment variables
cp .env.example .env

### Install project dependencies
npm install

### Initialize development server
npm run dev



## Installation client

### Install project dependencies
npm install

### Initialize development server
npm run develop || gatsby develop

### Run production
npm run build || gatsby build
```
